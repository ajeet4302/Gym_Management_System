# Gym_Management_System
A web-based Gym Management System developed using Java technologies to manage members, trainers, and workout plans efficiently. This application allows admin users to perform CRUD operations on members and trainers, view structured workout plans, and manage gym data through a user-friendly dashboard.
Features-
-Admin Login Authentication
-Add, View, Update, Delete Members
-Add, View, Update, Delete Trainers
-Dashboard with total members and trainers
-Workout Plans (Beginner, Intermediate, Professional)
-Clean and responsive UI with CSS

Technologies Used-
-Java (Servlets & JSP)
-Hibernate ORM
-Oracle Database
-Apache Tomcat Server
-HTML, CSS
